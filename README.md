# tourismguide
Chatbot for Tourism & Travel
Overview
This is an AI-powered chatbot designed to assist travelers by providing answers to common travel-related queries, personalized itinerary suggestions, and emergency assistance in case of cancellations or delays. The chatbot interacts dynamically with users and generates responses using the Google Gemini AI API.
